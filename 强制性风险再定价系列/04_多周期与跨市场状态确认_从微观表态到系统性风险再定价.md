# 第四篇：多周期与跨市场状态确认：从微观表态到系统性风险再定价

## 摘要

微观结构给入场，宏观结构给权限。单一品种的盘口可以告诉交易员局部是否出现被动风险调整流、吸收、接受或流动性空洞，但它不能单独说明这次变化是否属于系统性风险再定价。一个 1 分钟 Footprint 的下方吸收，不等于 4H 大反转；一个 NQ 的局部破位，也不等于整个风险资产进入去杠杆。多周期和跨市场确认的作用，是把局部表态放进更大的风险地图中。

本文多周期与跨市场状态确认模块，定义 4H、1H、30m、5m、1m / Footprint 的分工，说明微观信号的权限边界，建立指数期货跨市场确认清单，并讨论市场宽度、Open Interest / Positioning 和执行门禁。核心结论是：低周期信号可以触发执行，但高周期和跨市场状态决定仓位权限、路径权重和持有时间。

## 一、多周期状态一致性

多周期分析不是把多个图表叠在一起寻找共振，而是给不同周期分配不同职责。高周期定义地形，低周期定义入场；高周期提供风险方向和边界，低周期提供成交验证和执行触发。

| 周期 | 主要职责 | 适合回答的问题 | 不适合做什么 |
|---|---|---|---|
| 4H | 地形与主风险方向 | 市场处于上行、下行、压缩还是分配 | 不负责精确入场 |
| 1H | 成本带与价格发现阶段 | 旧成本锚是否迁移，新区域是否接受 | 不负责逐笔吸收 |
| 30m | 结构接受 / 拒绝 | 突破、收复、回测是否成立 | 不负责秒级止损 |
| 5m | 路径节奏 | 延续、回归、中继的节奏 | 不负责系统性判断 |
| 1m / Footprint | 执行触发 | 主动流、吸收、推进效率、LOB | 不负责大趋势定义 |

一个完整交易假设应写清楚：高周期允许什么方向，1H 和 30m 如何定义成本结构，5m 如何观察路径，1m / Footprint 如何执行。若没有这个分工，交易员容易用低周期噪音推翻高周期结构，或用高周期观点压制低周期失效。

## 二、微观信号的权限问题

微观信号的优势在于早，劣势在于局部。1 分钟下方吸收可以说明某一轮卖压在当前价位被承接，但它不能自动说明 4H 下跌趋势结束。它可能只是趋势中的短线回归、止盈、库存调整或低流动性修复。

| 微观信号 | 可以说明 | 不能直接说明 |
|---|---|---|
| Footprint 吸收 | 某价位主动流未能推进 | 高周期趋势反转 |
| CVD 背离 | 成交流与价格推进不一致 | 系统性风险结束 |
| LOB 撤单 | 当前流动性供给下降 | 基本面方向改变 |
| 1m 收复 | 局部偏离失败 | 日线成本锚失效 |

因此，微观信号需要权限。若高周期仍处于下行价格发现，1m 下方吸收可以作为短线回归交易，但不能默认做成波段反转。若 4H 已经进入深层 CARCB，1H 出现下行衰竭，30m 收复旧边界，5m 形成新成本带，此时 1m 吸收才可能获得更高权限。

微观信号权限的核心是：低周期证据只能证明低周期状态，除非高周期结构和跨市场变量也发生变化。

## 三、跨市场确认

强制性风险再定价常常不只发生在单一合约。指数期货尤其需要跨市场确认。NQ 的破位如果只有 NQ 自己发生，可能是科技权重、单一股票或局部流动性问题；若 ES、RTY、VIX、收益率、美元、credit spreads 和市场宽度同步确认，则更接近系统性风险再定价。

指数期货至少应观察以下变量：

| 变量 | 观察问题 | 风险含义 |
|---|---|---|
| ES / NQ / RTY | 指数期货是否同步 | 风险偏好是否扩散 |
| SPX / NDX options | Gamma、Skew、Call/Put Wall | 期权路径弹性 |
| VIX / VVIX | 波动率和波动率的波动率 | 尾部风险需求 |
| US10Y / US2Y yields | 利率冲击和曲线变化 | 折现率与宏观压力 |
| DXY | 美元流动性和避险 | 跨资产压力 |
| credit spreads | 信用风险 | 系统性风险偏好 |
| Mag7 / NVDA / AAPL / MSFT | 权重股驱动 | NQ / SPX 内部结构 |
| market breadth | 上涨/下跌扩散 | 趋势是否健康 |
| tick index / cumulative tick | 日内广度和资金流 | 短线风险偏好 |

跨市场确认不是要求所有变量都同向，而是判断单品种信号是否有外部支持。若关键外部变量冲突，仓位应降级；若外部变量同向，顺向路径权重提高。

## 四、跨市场确认与否定

示例性路径一：NQ 下破成本锚，但 ES 不确认，RTY 横盘，VIX 不动，US10Y 没有上行，DXY 稳定，Mag7 中只有单一权重股下跌。此时 NQ 破位可能是局部流动性事件或个股权重影响。若 1m 出现下方吸收并收复破位边界，短线回归路径可以研究；但系统性下行再定价不应被过早确认。

示例性路径二：NQ 下破成本锚，ES 同步跌破 RTH VWAP，RTY 更弱，VIX 上行，US10Y 上行，DXY 走强，Put Skew 走陡，market breadth 明显恶化。此时单品种破位获得跨市场确认，更可能是系统性风险再定价。若下方主动卖出被接受，仓位权限可以高于单一合约信号。

跨市场否定同样重要。若某商品期货因新闻上冲，但相关期限结构、现货代理变量和相关品种都不确认，上冲更可能被吸收。若利率期货异动，但美元、曲线和风险资产没有对应反应，需要警惕局部流动性或单一订单影响。

| 单品种信号 | 外部确认 | 权重处理 |
|---|---|---|
| 破位 | 外部变量同向 | 顺向路径权重提高 |
| 破位 | 外部变量不动 | 仓位降级 |
| 突破 | 外部变量冲突 | 等待收盘或新成本锚 |
| 吸收 | 外部变量支持反向 | 回归路径权重提高 |

## 五、市场宽度

市场宽度（Market Breadth）用于判断价格运动是否由少数权重推动，还是由整体风险偏好扩散。NQ 创新高但 breadth 弱，可能只是少数大型科技股推动；ES / NQ / RTY 同步，breadth 扩散，则说明风险偏好真实改善的概率更高。

| 宽度状态 | 价格表现 | 含义 |
|---|---|---|
| 指数上行，breadth 强 | 多数成分股参与 | 风险偏好扩散 |
| 指数上行，breadth 弱 | 权重股推动 | 上行脆弱 |
| 指数下行，breadth 恶化 | 广泛卖压 | 系统性压力 |
| 指数下行，breadth 稳定 | 局部权重拖累 | 破位需降权 |

收盘强度也要结合 breadth。高位强收若 breadth 扩散，更容易被解释为日终风险接受；高位强收若只由少数权重股支撑，次日若权重股回落，收盘确认的可靠性下降。低位弱收若 breadth 大幅恶化，则隔夜压力更可信；若 breadth 没有恶化，可能只是局部去杠杆。

## 六、Open Interest / Positioning

Open Interest（OI）和 positioning 数据未必适合日内执行，但对复盘和波段研究有价值。价格与 OI 的组合，可以帮助区分新仓进入、旧仓退出和回补行为。

| 价格 | OI | 可能含义 | 需要注意 |
|---|---|---|---|
| 上涨 | 上升 | 新多进入 / 新空对冲 | 需要看成交是否接受 |
| 上涨 | 下降 | 空头回补 | 回补结束后可能失速 |
| 下跌 | 上升 | 新空进入 / 新多被吸收 | 可能形成延续压力 |
| 下跌 | 下降 | 多头平仓 / 去杠杆 | 可能是风险释放 |

OI 的局限也很明显。日内 OI 未必实时可得，不同交易所发布时间不同，组合和跨品种对冲会使解释复杂。因此 OI 更适合进入盘后样本库，而不是作为秒级入场触发。

Positioning 还应结合 CFTC、基金流、期权持仓和期限结构等数据。若市场已经极端拥挤，价格进入 CARCB 后的强制调整可能更剧烈；若仓位较轻，突破可能缺少被动流燃料。

## 七、多周期与跨市场执行门禁

多周期和跨市场确认最终要进入执行门禁。

| 门禁条件 | 处理 |
|---|---|
| 高周期不支持，低周期信号成立 | 只能做短线，仓位降级 |
| 高周期支持，低周期未验证 | 等待执行触发 |
| 跨市场不确认 | 仓位降级，减少持有时间 |
| 系统性确认出现 | 顺向路径权重提高 |
| 外部变量冲突 | 不预设大趋势，等待收盘 |
| 高周期失效 | 退出原假设，不用低周期硬撑 |

执行上可以写成一句话：微观信号决定是否可以入场，高周期和跨市场决定入场可以承担多大风险、持有多久、失效条件放在哪里。若 1m 吸收与 4H 下行趋势冲突，止盈和仓位应按短线处理；若 1m 吸收、30m 收复、1H 新成本形成、4H 进入深层 CARCB，同时跨市场风险缓和，反转路径才有更高权限。

## 八、示例性路径三：低周期吸收但高周期不支持

示例性路径三：某指数期货 4H 处于下行价格发现，1H 成本锚不断下移，30m 反弹都无法收复旧边界。盘中价格在 5m 下方出现吸收，1m Footprint 显示主动卖出很大但价格不再创新低。短线交易员可以研究回到 5m VWAP 的回归，但不能把它定义为 4H 大反转。

若交易员按高周期反转处理，止盈会过远，止损也容易被放宽，最终把低周期交易变成无计划持仓。正确处理是：仓位小，目标短，失效条件明确；若后续 30m 收复旧边界、1H 成本带迁移，再升级路径。

## 九、示例性路径四：跨市场确认后的系统性风险再定价

示例性路径四：FOMC 后 NQ 跌破日内成本锚，ES 同步跌破 RTH VWAP，RTY 更弱，VIX 上行，VVIX 上行，US10Y 上行，DXY 走强，market breadth 快速恶化，Put Skew 走陡。此时 NQ 的破位不再只是单品种微观信号，而是跨市场风险状态变化。

若价格反弹不回旧成本锚，并在下方形成新成交，交易员可以把路径定义为系统性风险再定价中的下行接受。执行仍不能追最薄盘口，但顺向路径权重和持有权限高于单品种破位。失效条件是跨市场变量缓和、价格收复旧成本锚、下方卖压被吸收。

## 十、多周期样本记录表

多周期和跨市场确认需要被记录，否则容易沦为主观筛选。

| 字段 | 记录内容 |
|---|---|
| 4H 状态 | 趋势、压缩、深层偏离、成本锚 |
| 1H 状态 | 主成本带、新旧锚关系 |
| 30m 状态 | 接受、拒绝、收复、回测 |
| 5m 路径 | 延续、回归、中继、衰竭 |
| 1m / Footprint | 主动流、吸收、推进效率 |
| 跨市场确认 | ES/NQ/RTY、VIX、收益率、DXY、breadth |
| 执行权限 | 标准、降级、短线、禁止 |

复盘时应统计：高周期支持时低周期信号表现如何，跨市场确认缺失时追随交易是否失效，breadth 背离是否降低收盘强度的延续概率。只有长期样本，才能判断哪些确认变量真正有价值。

### 多周期权重：从方向到执行权限

多周期分析最常见的误用，是把所有周期都拿来证明同一个观点。研究型交易必须反过来处理：每个周期只回答自己有能力回答的问题。高周期决定背景和风险预算，中周期决定成本锚和路径结构，低周期决定触发与失效。若 1 分钟图出现吸收，它只能说明局部价位的主动流暂时无法推进；它不能单独推翻 4 小时级别的下行成本锚，也不能证明系统性风险结束。

| 周期组合 | 状态含义 | 执行权限 |
|---|---|---|
| 高周期支持，低周期验证 | 背景、路径和触发一致 | 标准权限，仍需控制滑点 |
| 高周期支持，低周期未验证 | 观点有背景，但盘口未表态 | 等待，不提前执行 |
| 高周期冲突，低周期验证 | 局部机会，不是状态迁移 | 仓位降级，缩短持有 |
| 高周期中性，低周期验证 | 短线路径成立 | 只按局部失效管理 |
| 高周期支持但跨市场否定 | 单品种结构有瑕疵 | 降级或等待收盘 |
| 多周期和跨市场同时确认 | 状态切换权重提高 | 可研究延续与中继 |

这张表的核心不是提高交易频率，而是限制解释范围。交易员如果在低周期上看到强吸收，却把它写成“日线反转”，就是权限越界；如果宏观观点很强，却在订单簿已经否定的情况下继续持仓，也是权限越界。多周期框架的价值，是让每个证据承担有限责任。

### 跨市场确认强度分级

跨市场确认也需要分级。并不是多看几个品种就能提高判断质量。真正有用的是确认变量与当前交易假设之间存在机制关联：股指下跌若伴随 VIX 上行、收益率冲击、美元走强和市场宽度恶化，说明风险偏好可能发生系统性再定价；若只有某一指数下跌，而宽度稳定、VIX 不动、权重股结构分化，则更可能是局部权重或流动性问题。

| 确认等级 | 条件 | 处理 |
|---|---|---|
| 0 级：无确认 | 只有单一合约动作，外部变量无反应 | 只按局部路径处理 |
| 1 级：弱确认 | 同板块或近邻合约轻微同向 | 仓位降级，等待主时段 |
| 2 级：中等确认 | 相关资产同向，波动率或宽度配合 | 顺向路径权重提高 |
| 3 级：强确认 | 利率、美元、波动率、宽度或期限结构共同支持 | 可研究系统性再定价 |
| 冲突确认 | 价格方向与核心外部变量相反 | 暂停放大解释 |

分级的好处，是把“确认”从模糊词变成复盘字段。后续样本可以检验：3 级确认的延续概率是否高于 0 级确认；冲突确认下的追随交易是否更容易出现 MAE 扩大；强确认是否也会带来更大滑点和更高的执行成本。确认不是只看胜率，还要看风险路径。

### 失效细节：确认如何被撤销

多周期和跨市场确认不是一次性标签。它会被后续交易撤销。高周期支持可以因为旧成本锚收复而失效；跨市场确认可以因为外部变量缓和而失效；低周期执行可以因为吸收失败或流动性空洞而失效。若交易员不记录确认撤销条件，框架会变成不断给原观点加理由的工具。

| 原确认 | 撤销信号 | 研究处理 |
|---|---|---|
| 高周期下行 | 价格收复 1H 主成本带并形成新 POC | 下行路径降权 |
| 低周期卖压接受 | CVD 继续下降但价格不再下跌，出现补单 | 研究吸收 |
| VIX 上行确认风险 | VIX 回落、VVIX 不跟、股指收复 VWAP | 系统性压力降级 |
| 美元/利率确认宏观压力 | DXY 或收益率反向收复事件区 | 宏观链条失效 |
| 宽度恶化确认下跌 | 指数弱但 breadth 修复 | 指数破位降权 |

失效细节尤其重要，因为跨市场变量有时会先于标的缓和。若 NQ 仍在低位，但 VIX 已经回落、收益率停止上行、breadth 不再恶化，下行延续的质量就下降。此时继续用最初的强确认解释持仓，属于样本外延伸。

### 示例性路径五：单品种强、跨市场弱的上行突破

示例性路径：某日 NQ 在大型权重股推动下突破上午高点，1 分钟 Footprint 显示主动买入有效，旧高回测也防守成功。低周期看，这是合格突破。但同时 ES 只是略高于 VWAP，RTY 仍在日内中位，market breadth 没有扩散，VIX 不跌，DXY 和收益率没有提供风险偏好支持。此时应把路径定义为“单品种上行接受，但系统性确认不足”。

交易处理上，低周期多头并非完全无效，但持有权限需要降级。更合理的计划是：只按 NQ 新成本带管理，不把它升级为全市场风险偏好扩张；若权重股回落或 NQ 跌回突破区，立即撤销上行状态；若随后 ES、RTY、breadth 和 VIX 开始确认，再把路径从局部突破升级为跨市场风险再定价。这个例子说明，跨市场框架不是为了否定每个单品种机会，而是为了防止交易员把局部机会当作系统性趋势。

### 多周期复盘的最低样本要求

为了让多周期判断可证伪，样本记录至少要保留三个版本的结论。第一，盘前高周期结论，记录主成本锚、压缩/扩张状态和跨市场背景。第二，盘中低周期结论，记录具体执行触发、主动流、LOB 和失效价位。第三，盘后整合结论，记录高低周期是否一致、跨市场是否确认、原假设是否被撤销。若只保留盘后总结，研究者很容易把所有周期重新排列成一个事后完美故事。

复盘时可以设置一个简单评分：高周期 0-2 分，低周期验证 0-2 分，跨市场确认 0-2 分，收盘强度 0-2 分，执行质量 0-2 分。总分不是入场信号，而是样本分组工具。统计一段时间后，交易员可以比较高分样本是否真的比低分样本更少出现假突破，是否有更低 MAE，是否更适合隔夜持有。若没有差异，评分标准就需要被修正。

### 延迟确认与提前否定

跨市场变量并不总是同步表态。有些时候，标的合约先突破，波动率和宽度稍后确认；有些时候，外部变量先缓和，标的价格仍在惯性延续。研究型交易不能要求所有变量在同一分钟同时完成确认，但必须定义等待期限和提前否定条件。否则，任何延迟都可能被解释为“确认还没来”，任何冲突都可能被拖延到失去意义。

| 情形 | 处理方式 | 风险 |
|---|---|---|
| 标的先动，外部变量 5-15 分钟后确认 | 可先按低周期短线处理，确认后提高权限 | 过早按系统性路径持有 |
| 外部变量先动，标的尚未破位 | 只做预警，不提前入场 | 把宏观线索当作触发 |
| 标的延续，VIX / breadth 提前缓和 | 顺向仓位降级，等待价格确认 | 忽略风险偏好修复 |
| 利率、美元与股指信号冲突 | 缩短持有时间，等待收盘 | 选择性使用变量 |

延迟确认的关键是时间上限。若一个 NQ 下破在 30 分钟后仍没有 ES、RTY、VIX、breadth 或利率变量支持，就不应继续把它描述为系统性风险再定价；它最多是单品种路径。反过来，如果跨市场变量先给出风险压力，但标的一直没有离开成本锚，也不能提前把宏观压力变成期货仓位。确认框架既要允许市场不同步，也要防止无限等待。

反例路径：1 分钟 Footprint 在低位出现明显卖压吸收，价格短暂收复局部低点。但 4H 仍处在下行价格发现中，30m 回撤无法收复前一段低点，ES、RTY 和市场宽度继续走弱，VIX 没有回落。这个样本中的低周期吸收可以支持日内短线回归或止盈管理，但不能升级为高周期反转。多周期框架的作用，是限制低周期证据的权限。

## 十一、研究边界

有公开资料和市场结构支持的部分，是不同周期承载不同信息，跨资产变量会共同反映风险偏好、利率冲击、美元流动性和波动率需求。可观察但不能直接确认的部分，是某一跨市场同步究竟来自同一资金流、宏观信息、套利机制还是流动性撤出。需要事件研究验证的交易假设，是多周期一致性和跨市场确认是否改善路径识别、降低假突破、提高风险收益。多周期和跨市场确认不是为了制造更多理由，而是为了限制低周期信号的权限。

## 十二、小结

微观结构给入场，宏观结构给权限。没有微观验证，高周期观点容易变成主观叙事；没有高周期和跨市场确认，微观信号容易被过度放大。强制性风险再定价需要同时回答两个问题：局部市场是否已经表态，系统风险是否允许这个表态被放大。

交易员真正要避免的，是用 1m 信号讲 4H 故事，或用宏观观点忽略盘口失效。多周期和跨市场框架的价值，就在于让每个证据待在自己的权限范围内。
