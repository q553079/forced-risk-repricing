# 强制性风险再定价系列

## 项目简介

本仓库整理的是一组关于期货市场“强制性风险再定价”（Forced Risk Repricing）的中文研究型交易长文。系列目标不是建立静态点位交易法，也不是给出收益承诺，而是把制度约束、成本锚、信息冲击、时间结构、期权对冲、异质参与者、交易流与限价订单簿状态串联成一个可验证的市场观察框架。

本系列关注的不是价格变化本身，而是风险约束被激活后的状态切换：部分市场参与者在保证金、成本、风险预算、期权对冲、库存、止损、清算或流动性约束下，无法继续保持原有风险状态，必须通过交易、对冲、减仓、平仓、撤出流动性或重新定价来完成风险调整。

## 核心命题

价格变化本身并不天然具有交易意义。真正值得研究的，是风险约束被激活后的强制性状态切换。

核心链条：

> 制度制造约束，成本提供锚点，偏离激活压力，信息冲击改变预期分布，期权改变对冲流，时间结构确认风险状态，跨市场关系检验系统性，压缩积累风险，扩张迫使表态，交易流与限价订单簿状态给出验证。交易利润来自对这些状态切换的识别、验证与执行。

## 核心术语表

| 术语 | 英文 | 简要说明 |
|---|---|---|
| 强制性风险再定价 | Forced Risk Repricing | 风险约束被激活后，参与者必须调整仓位、对冲、库存或流动性供给的状态切换 |
| 成本锚 | Cost Anchor / Transaction Cost Anchor | 用于测量价格偏离、盈亏压力和风险约束的成交成本参照 |
| 成本锚定风险约束带 | Cost-Anchored Risk Constraint Band, CARCB | 围绕成本锚展开的分层风险约束区域 |
| 被动风险调整流 | Forced Risk-Adjustment Flow | 由止损、保证金、对冲、风控或清算约束触发的非自愿交易流 |
| 交易流与限价订单簿状态 | Trade Flow and Limit Order Book State | 用主动成交、深度、撤单、补单和吸收判断市场承接状态 |
| 价格接受 | Price Acceptance | 新价格区域被持续成交、回撤浅且形成新成本区 |
| 价格拒绝 | Price Rejection | 新价格区域无法维持，价格收复旧边界或回到旧成本锚 |
| 波动率压缩 | Volatility Compression | 成交、止损、库存和期权对冲需求在窄区间内聚集 |
| 价格发现扩张 | Price Discovery Expansion | 价格离开旧成本锚，测试新区域是否被市场接受 |
| 事件研究 | Event Study | 用结构化字段记录同类事件，检验路径假设和执行规则 |

## 如何使用本系列

本系列适合作为研究框架、复盘模板和交易计划的语言系统使用。读者可以把文章中的成本锚、CARCB、期权环境、交易流、LOB 状态、收盘强度和跨市场确认转化为盘前计划、盘中观察清单和盘后事件研究字段。

不建议把任何单一概念直接当成入场信号。成本锚不是支撑压力，CARCB 不是反转带，Call Wall / Put Wall 不是方向预测，主动成交净差也不是单独信号。框架的使用顺序应是：先定义旧状态，再提出约束假设，然后等待成交与流动性验证，最后以可复盘的失效条件进入执行。

## 研究边界

本系列区分三类内容：

- 公开制度证据支持的部分：保证金制度、清算机制、SPAN / SPAN 2、期权对冲逻辑、市场微观结构理论。
- 可观察但不能直接确认身份的部分：参与者行为、dealer hedge、HFT 撤流动性、被动风险调整流。
- 需要事件研究验证的部分：CARCB 层级、成本锚选择、路径概率、交易胜率、止损距离、品种适配。

所有示例性路径均用于说明研究框架，不伪装成真实历史事件。本系列的有效性必须接受样本统计、事件研究、逐笔复盘和真实执行结果检验。
