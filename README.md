# 强制性风险再定价系列

本仓库整理的是一组关于期货市场“强制性风险再定价”（Forced Risk Repricing）的中文研究型交易长文。系列目标不是建立静态点位交易法，也不是承诺收益，而是把制度约束、成本锚、信息冲击、时间结构、期权对冲、异质参与者、交易流与限价订单簿状态串联成一个可验证的市场观察框架。

核心链条：

> 制度制造约束，成本提供锚点，偏离激活压力，信息冲击改变预期分布，期权改变对冲流，时间结构确认风险状态，跨市场关系检验系统性，压缩积累风险，扩张迫使表态，交易流与限价订单簿状态给出验证。交易利润来自对这些状态切换的识别、验证与执行。

## 研究边界

本系列区分三类内容：

- 公开制度证据支持的部分：保证金制度、清算机制、SPAN / SPAN 2、期权对冲逻辑、市场微观结构理论。
- 可观察但不能直接确认身份的部分：参与者行为、dealer hedge、HFT 撤流动性、被动风险调整流。
- 需要事件研究验证的部分：CARCB 层级、成本锚选择、路径概率、交易胜率、止损距离、品种适配。

所有示例性路径均用于说明研究框架，不伪装成真实历史事件。本系列的有效性必须接受样本统计、事件研究、逐笔复盘和真实执行结果检验。
